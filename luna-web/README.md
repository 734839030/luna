#luna-web 介绍
> luna-web 包含一系列web 相关封装

* 统一异常处理
* 统一参数校验
* 统一响应体，错误码规范


### Maven dependency

```
<dependency>
	<groupId>com.seezoon</groupId>
	<artifactId>luna-web</artifactId>
	<version>${luna.version}</version>
</dependency>
```
