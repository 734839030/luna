#luna-service 介绍
> luna-service 包含一系列service 相关封装

* 封装增删该查
* 实体规范



### Maven dependency

```
<dependency>
	<groupId>com.seezoon</groupId>
	<artifactId>luna-service</artifactId>
	<version>${luna.version}</version>
</dependency>
```
